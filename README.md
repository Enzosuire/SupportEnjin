# Support Enjin

Application de support technique réalisée avec **FreeScout** pour la plateforme Enjin.

## Description

Ce projet vise à centraliser et gérer l’assistance aux utilisateurs de la plateforme Enjin. Il permet :

- La gestion et le suivi des tickets utilisateurs  
- La résolution de problèmes techniques  
- La communication efficace avec les utilisateurs  
- L’optimisation des workflows de support  

## Stack technique

- **Interface** : FreeScout  
- **Technologies** : Laravel, PHP, MySQL, HTML/CSS, Bootstrap, JavaScript  
- **Fonctionnalités clés** : Gestion des tickets, notifications, suivi des interventions  

## Objectif

Création de différents modules spécifiques, tels que des modules de facturation, de gestion des projets, de suivi des interventions, ainsi qu’une base de connaissances et des améliorations spécifiques pour la plateforme Enjin. 

Fournir un support structuré et efficace pour la plateforme Enjin, améliorant la satisfaction des utilisateurs et la réactivité de l’équipe technique.
